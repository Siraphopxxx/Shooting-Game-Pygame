# JHON IS MAN
- A Retro-style 2D Side-scroller Shooter built with Python & Pygame.

- JHON IS MAN เป็นเกมแนว Action Platformer ที่ผู้เล่นจะได้รับบทเป็น "Jhon" ทหารกล้าที่ต้องฝ่าด่านศัตรู (Bots) พร้อมบริหารจัดการทรัพยากรอย่างกระสุนและพลังชีวิต

# ฟีเจอร์ภายในเกม
- Tile-based World: ระบบแผนที่ที่สร้างจาก Grid Data (0, 1, 2) ทำให้การออกแบบด่านทำได้ง่าย

- Smart Bot AI: ศัตรูที่มีระบบตรวจจับระยะ (Proximity Sensing) และจะทำการยิงต่อเมื่อผู้เล่นอยู่ในระดับสายตา

- Interactive Items: ระบบกล่อง Item (Health & Ammo) ที่มีการใช้ Sprite Collision ในการตรวจจับการเก็บไอเทม

- Dynamic UI: แถบ HP Bar แบบ Real-time ทั้งของผู้เล่นและศัตรู (Individual Bot Health Tracking)

- Full Audio Experience: ระบบเสียงประกอบครบวงจร ทั้งเสียงเดินบนหญ้า, เสียงยิงปืน และเพลงประกอบฉาก

# วิธีการเล่น
- W: กระโดด (Jump)

- A / D: เคลื่อนที่ ซ้าย/ขวา

- Spacebar: ยิงปืน (Shoot)

- R: เริ่มเกมใหม่ (Restart) / Q: ออกจากเกม (Quit) เมื่อจบเกม
